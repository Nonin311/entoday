# grid_flowchart_designer
这是一个用于制作流程图的代码，配置了相关UI界面用于人工操作，同时在设计的时候也考虑了被其他代码调度的功能，以到达一些自动化的目的。翻译成英文。
This is a tool for creating flowcharts. It features a user interface (UI) for manual operation, and is also designed to be programmatically invoked by other code to facilitate automated workflows.
对于手动操作部分，双击UI界面上的框图按钮可以创建对应的框图（坐标合法且随机），依次选择两个端口可以创建连线。
For manual operations, double-clicking a block button on the UI creates the corresponding block (with valid, randomized coordinates). Selecting two ports in sequence will create a connection line.
